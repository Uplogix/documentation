To connect the Uplogix appliance to an IBM AIX serial port, use the following pinout:

|RJ-45	|Color	|DB-9|
|---|---|---|
|1	|Blue	|7|
|2|	Orange	|4|
|3|	Black	|3|
|4|	Red	|5|
|5|	Green	|5|
|6|	Yellow	|2|
|7|	Brown	|1|
|8|	White	|8|

See the attached picture for a view of the pin-out.
