Clears the password on a device. Rules can be defined to automate this operation using the *clearPassword* action.

# Command availability 

CLI resource: port

Device makes: Juniper

Uplogix systems: All

LMS offerings: All

# Syntax 

```
clear password
```

# Usage 

```
[admin@UplogixLM (port1/1)]# clear password
Password cleared
```
